Wine Quality Prediction

Predict the quality of wine using Machine Learning

About This Project

This application uses advanced Machine Learning algorithms to predict the quality of wine based on its physicochemical properties. Simply enter the wine's characteristics and get an instant quality prediction!

Input Features

  • Fixed Acidity - Primary acids in wine
  • Volatile Acidity - Acetic acid content
  • Citric Acid - Freshness enhancer
  • Residual Sugar - Remaining sugar after fermentation
  • Chlorides - Salt content
  • Free Sulfur Dioxide - SO₂ preservative (free form)
  • Total Sulfur Dioxide - Total SO₂ content
  • Density - Wine density
  • pH - Acidity level
  • Sulphates - Wine additives
  • Alcohol - Alcohol percentage
